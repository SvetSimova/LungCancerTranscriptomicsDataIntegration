##Data Description
##==========================================
## GTEx data https://www.gtexportal.org/home/

Normal lung tissue

GTEx_Lung.csv - gene expression from normal lung. Genes in rows, samples in column. Sample ID contains patient ID
GTEx_patients.csv - info about patients, patients in rows
	$ SUBJID : patient name, is included in sample name ID
	$ SEX    : 1-male, 2-female
	$ AGE    : chr  "60-69" "50-59" "60-69" "60-69" ...
	$ DTHHRDY: int  4 0 1 3 0 0 2 NA 2 2 ...


##==========================================
## TCGA data https://portal.gdc.cancer.gov/

Tumour and normal lung tissue from 2 TCGA datasets: LUAD (adenocarcinoma) and LUSC (squamous cell carcinoma)

TCGA_Lung.csv - gene expression from cancer lungs. Genes in rows, samples in column. Sample ID contains patient ID
TCGA_samples.csv  - info about samples, samples in rows. Some samples from the same patients - patient code: TCGA-xx-xxxx
	$ SAMPLEID : sample ID, same as column in TCGA_Lung.csv
	$ DATASET  : study name, LUAD or LUSC
	$ SAMPLE   : histologically normal an cancer sample
	$ HISTOLOGY: specific histology of the tumour
	$ LOCATION : location of the biopsy 
	$ SEX      : sex of the patients
	$ AGE      : sex of the patients at diagnostics
	$ TIMESURV : time of survival in days
	$ EVENTSURV: event happened as "TIMESURV": "alive" - left the study (censored) "dead" - dead
